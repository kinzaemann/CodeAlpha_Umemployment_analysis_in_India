CodeAlpha Unemployment Analysis
## 📌 Project Overview
This project focuses on analyzing the unemployment trends in India, specifically highlighting the impact of the COVID-19 pandemic. Using Python, I performed data cleaning, exploratory data analysis (EDA), and created interactive visualizations to identify regional and seasonal patterns in the labor market.

## 🛠️ Technologies Used
* **Python**: Core programming language.
* **Pandas**: For data cleaning and manipulation.
* **Plotly**: For interactive visualizations (Line charts, Choropleth maps).
* **Matplotlib & Seaborn**: For statistical analysis and heatmaps.

## 📈 Key Insights & Objectives
* **Analyze COVID-19 Impact**: Identified a significant surge in unemployment exceeding 20% during the 2020 lockdown.
* **Regional Disparities**: Used Choropleth maps to visualize how different states (like Haryana and Tripura) were affected differently.
* **Urban vs Rural**: Compared the volatility between city-based and village-based employment.
* **Seasonal Trends**: Analyzed monthly fluctuations to identify economic anomalies.

## 📂 Repository Structure
* `unemployment_analysis.ipynb`: The main Python notebook containing all code and visualizations.
* `dataset.csv`: The raw dataset used for analysis.
* `requirements.txt`: List of Python libraries required to run the project.
* `README.md`: Project documentation and summary.

### 📈 Key Findings: The COVID-19 Impact
* **Pre-Lockdown Average:** 9.51% 
* **Post-Lockdown Average:** 17.77%
* **Net Increase:** 86.91% surge in unemployment rates.
* **Insight:** This nearly 87% spike illustrates the high vulnerability of the Indian labor market to sudden industrial halts.

  
## 🏁 Conclusion
The analysis successfully demonstrates the labor market's sensitivity to movement restrictions. These insights could inform economic policies, such as targeted relief for high-risk urban centers during future crises.
